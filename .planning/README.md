# CLI命令解释Agent - 项目规划

## 项目概述

开发一个智能Agent，帮助用户理解CLI工具（如Claude Code、GitHub Copilot CLI等）请求执行的命令，提供：
1. **命令解释**：详细说明命令的作用
2. **风险评估**：评估命令可能造成的影响
3. **建议决策**：帮助用户判断是否执行

---

## 📌 分阶段开发策略

本项目采用**两阶段开发**，确保快速交付可用产品，同时为未来扩展预留空间。

### Phase 1：快捷键 + 剪贴板模式（MVP）
> **目标**：快速实现核心功能，验证AI解释与风险评估的价值

- **触发方式**：用户复制命令 → 按下快捷键 → Agent解释命令
- **开发周期**：约 1-2 周
- **状态**：✅ 当前阶段

### Phase 2：终端代理模式（可选进阶）
> **目标**：实现自动监测，无需手动复制

- **触发方式**：用户通过 `agent run <cmd>` 启动CLI工具，Agent自动捕获输出
- **开发周期**：约 2-3 周（在Phase 1完成后）
- **状态**：🔜 待评估需求后启动
